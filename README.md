# E-Commerce-Analysis----Amazon
Amazon veri setini dbt, BigQuery ve Looker Studio kullanarak analiz ettiğim bu projede; ham veriler dbt ile dönüştürülmüş, satış ve fiyat verileri birleştirilerek gelir, kâr oranı, kategori ve kanal bazlı performans metrikleri hesaplanmıştır. Elde edilen sonuçlar Looker Studio üzerinde görselleştirilerek etkileşimli raporlar haline getirilmiştir.
